# Plot MCMC trajectories and posterior distributions

This function uses the output of
[rjags::jags.model](https://rdrr.io/pkg/rjags/man/jags.model.html) to
visualise the traces of the MCMC and the corresponding densities. In
particular it displays the posterior distributions of the age, if it is
calculated, palaeodose and the equivalent dose dispersion parameters of
the sample. The function output is very similar to plot output produced
with the 'coda' package, but tailored to meet the needs in the context
of the `'BayLum'` package.

## Usage

``` r
plot_MCMC(
  object,
  sample_names = NULL,
  variables = c("A", "D", "sD"),
  axes_labels = c(A = "Age (ka)", D = "D (Gy)", sD = "sD (Gy)"),
  n.chains = NULL,
  n.iter = 1000,
  smooth = FALSE,
  rug = TRUE,
  plot_single = FALSE,
  ...
)
```

## Arguments

- object:

  [coda::mcmc.list](https://rdrr.io/pkg/coda/man/mcmc.list.html) or
  [coda::mcmc](https://rdrr.io/pkg/coda/man/mcmc.html) (**required**):
  Output generated by
  [rjags::jags.model](https://rdrr.io/pkg/rjags/man/jags.model.html),
  e.g., in
  [Age_Computation](https://crp2a.github.io/BayLum/dev/reference/Age_Computation.md).
  Alternatively, limited support is provided for `BayLum.list` object
  input

- sample_names:

  [character](https://rdrr.io/r/base/character.html) (optional): Names
  of the used samples. This argument overrides the optional argument
  `mtext`.

- variables:

  [character](https://rdrr.io/r/base/character.html) (with default):
  Variables in your [coda::mcmc](https://rdrr.io/pkg/coda/man/mcmc.html)
  object to be plotted.

- axes_labels:

  [character](https://rdrr.io/r/base/character.html) (with default):
  Axes labels used for the trace and density plots. The labels should be
  provided as named [character](https://rdrr.io/r/base/character.html)
  [vector](https://rdrr.io/r/base/vector.html) with the parameter names
  as the names used to assign the axes labelling. The labelling for the
  x-axis (trace plots) and y-axis (density plot) cannot be modified.

- n.chains:

  [numeric](https://rdrr.io/r/base/numeric.html) (optional): Set the
  number of chains to visualise, if nothing is provided the number of
  chains is determined from the input object

- n.iter:

  [integer](https://rdrr.io/r/base/integer.html) (with default): Set the
  number of iterations to be visualised in the trace plots, regardless
  of the size of the input dataset as long as the real number of
  iterations is \> `n.iter`. Please note that large numbers impact the
  plot performance.

- smooth:

  [logical](https://rdrr.io/r/base/logical.html) (with default):
  Enable/disables smooth of trace plots using
  [stats::smooth](https://rdrr.io/r/stats/smooth.html)

- rug:

  [logical](https://rdrr.io/r/base/logical.html) (with default):
  Enable/disables [rug](https://rdrr.io/r/graphics/rug.html) under
  density plots

- plot_single:

  [logical](https://rdrr.io/r/base/logical.html) (with default):
  Enables/disables the single plot mode of the function, i.e. if set to
  `TRUE` every plot is returned in a single plot and own
  [par](https://rdrr.io/r/graphics/par.html) settings can be applied.

- ...:

  further arguments that can be passed to modify the plot output.
  Supported arguments are `lwd`, `lty`, `col`, `type`, `cex`,`mtext`,
  cf. [mtext](https://rdrr.io/r/graphics/mtext.html) for `mtext` and
  [plot.default](https://rdrr.io/r/graphics/plot.default.html) for the
  other arguments.

## Value

Two plots: Traces of the MCMC chains and the corresponding density
plots. This plots are similar to
[coda::traceplot](https://rdrr.io/pkg/coda/man/traceplot.html) and
[coda::densplot](https://rdrr.io/pkg/coda/man/densplot.html).

## Details

The function is used in the function
[Age_Computation](https://crp2a.github.io/BayLum/dev/reference/Age_Computation.md),
[AgeS_Computation](https://crp2a.github.io/BayLum/dev/reference/AgeS_Computation.md)
and
[Palaeodose_Computation](https://crp2a.github.io/BayLum/dev/reference/Palaeodose_Computation.md),
but can be used also as standalone plot function.
